# mafengwo-seeker
用于通过爬虫实时获取蚂蜂窝网站上用户关于中国景点的评论数据和游记数据，将前50页数据下载成excel文件，并通过简单的评价算法将评论数据和游记数据转化为景点热度，
并通过百度地图api将热度信息实时的展现出来。
